# Aplicación OCR y NLP para Certificados Médicos

## Descripción
Aplicación desarrollada en Python para extraer información de certificados médicos utilizando técnicas de OCR y procesamiento de lenguaje natural (NLP).

---

## Tecnologías utilizadas
- Python
- Streamlit
- OpenCV
- EasyOCR
- spaCy
- NLTK
- Pandas

---

## Funcionalidades
- Carga de imágenes
- Extracción automática de texto
- Procesamiento NLP
- Exportación de resultados en Excel y CSV

---

## Archivos principales

- `app4.py` → aplicación Streamlit
- `Parcial.ipynb` → notebook de desarrollo
- `requirements.txt` → librerías necesarias

---
